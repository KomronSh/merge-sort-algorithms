## 🔧 Решение распространенных проблем

### Проблема: "fatal: not a git repository"

```powershell
cd "F:\study\24ва1\СиАКОД\Курсовая работа"
# или
cd merge-sort-algorithms
```

### Проблема: Программа не компилируется

**Убедитесь:**
1. Visual Studio установлена с поддержкой C++
2. Выбран правильный язык проекта (C++)
3. Стандарт C++ установлен на C++17 или выше

**Проверка в Visual Studio:**
- Project Properties → C/C++ → Language Standard → установите ISO C++17 Standard (/std:c++17)

### Проблема: "Файл не найден" при запуске сортировки

Сначала создайте входной файл (опция 1 в меню).

### Проблема: Программа зависает при работе с большим файлом

- Прервите выполнение (Ctrl+C)
- Попробуйте с меньшим размером файла
- Проверьте оперативную память компьютера

### Проблема: Результаты отличаются от документации

Это нормально! Результаты зависят от:
- Размера входного файла
- Ваших сгенерированных случайных чисел
- Характеристик вашего компьютера

