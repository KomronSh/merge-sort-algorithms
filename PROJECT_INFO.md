## 🚀 Как загрузить на GitHub

### Способ 1: Используя HTTPS

```powershell
cd "F:\study\24ва1\СиАКОД\Курсовая работа"

# 1. Создайте репозиторий на GitHub.com
# 2. Скопируйте его URL (https://github.com/USERNAME/repository.git)
# 3. Выполните:

git remote add origin https://github.com/USERNAME/repository.git
git branch -M main
git push -u origin main
```

### Способ 2: Используя GitHub CLI

```powershell
# Установите GitHub CLI если еще не установлена
# https://cli.github.com/

cd "F:\study\24ва1\СиАКОД\Курсовая работа"
gh repo create merge-sort-algorithms --public --source=. --remote=origin --push
```

### Способ 3: Через SSH (если настроен)

```bash
git remote add origin git@github.com:USERNAME/merge-sort-algorithms.git
git branch -M main
git push -u origin main
```

## 💾 Git команды для работы после загрузки

### После загрузки на GitHub, используйте для обновлений:

```powershell
# 1. Внесите изменения в файлы
# 2. Проверьте статус
git status

# 3. Добавьте изменения
git add .

# 4. Создайте коммит
git commit -m "Описание изменений"

# 5. Загрузите на GitHub
git push
```

### Полезные команды:

```bash
# Просмотреть удаленные репозитории
git remote -v

# Обновить локальный репозиторий
git pull

# Создать новую ветвь для функции
git checkout -b feature/new-feature

# Переключиться на другую ветвь
git checkout main

# Объединить ветви
git merge feature/new-feature

# Посмотреть историю
git log --oneline --graph
```
